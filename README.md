# Birds-Watching-RUS

Неофициальный русификатор для игры *Birds Watching*.

## Скачать
Скачивание доступно в разделе **Releases** (справа).

---

## Установка

1. Скачайте установщик из раздела Releases
2. Запустите `.exe` файл
3. Укажите путь к папке с игрой (где находится `Birds Watching.exe`)
4. Дождитесь завершения установки

---

### Важно

Один из антивирусных движков может помечать файл как: Suspicious.low.ml.score
Это связано с тем, что:
- установщик новый
- изменяет файлы игры
- требует права администратора

Это **ложное срабатывание**.

Остальные проверки угроз не обнаружили.

---

## Что сделано

- Полный перевод текста игры
- Перевод диалогов (Dialogic)
- Перевод интерфейса
- Частичный перевод текстур
- Исправление структуры диалогов для избежания зависания в игре. Баг существует с самого начала игры, разработчик еще не выпустил официальный патч, решил исправить самостоятельно, сделал wait_time с перехода кат-сцены и диалога, что помгло избежать зависания в text-box.

---

## Возможные проблемы

Я проверил всё в переводе, перевод не ломает код игры, баги, которые встречаются связаны только с разработкой игы.

---

## 🧠 О переводе

Игра имеет нестандартную структуру:
- текст разбросан по `.csv`, `.dtl`, `.tscn`
- часть ресурсов находится в `.assets`
- использован кастомный подход к диалогам (Dialogic)

Для перевода использовался реверс-инжиниринг ресурсов игры.
